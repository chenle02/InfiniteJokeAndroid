# Infinite Joke Android

A tiny Android app version of the Infinite Joke Generator.

## Build APK on GitHub

1. Open this repository on GitHub.
2. Go to **Actions**.
3. Select **Build Android APK**.
4. Tap **Run workflow**.
5. Wait until the run finishes.
6. Open the finished run.
7. Download the artifact named **InfiniteJoke-debug-apk**.
8. Unzip it.
9. Install `app-debug.apk` on your Android phone.

## Expected size

The APK should be much smaller than **0.01 GB**.

## Notes

This is a real Android app using a native Android WebView. It opens as its own app, not as a normal browser tab.
